    Central Processing Unit (CPU):

    • Function: The CPU, often called the "brain" of the computer, is responsible for executing instructions from programs and performing calculations. It processes data and carries out tasks required to run applications and the operating system. The CPU interprets and executes commands from the computer's hardware and software
      • Key Features:
    • Cores: Modern CPUs contain multiple cores (dual, quad, octa), each capable of processing data independently, allowing for better multitasking.
    • Clock Speed: Measured in gigahertz (GHz), higher clock speeds indicate a faster CPU.
    • Popular Brands: Intel (Core i3, i5, i7, i9), AMD (Ryzen series).
    • Examples: Intel Core i7-11700K, AMD Ryzen 9 5900X

    Types of Motherboard (Form Factors):

    • ATX: The standard size for most desktop PCs. Supports a wide range of features, including multiple expansion slots and high RAM capacity. Dimensions: 305mm x 244mm (12" x 9.6").
    • MicroATX (mATX): Smaller than ATX but still supports most of the same features, although with fewer expansion slots. Ideal for mid-sized desktops or users who donâ€™t need as many PCIe slots. Dimensions: 244mm x 244mm (9.6" x 9.6").
    • Mini-ITX: Compact size, designed for small form-factor (SFF) PCs. Supports fewer RAM slots and expansion options but is perfect for portable or space-saving setups. Dimensions: 170mm x 170mm (6.7" x 6.7").
    • Extended ATX (E-ATX): Larger than standard ATX, used for high-end systems that require more PCIe slots, additional RAM capacity, and more power delivery options. Suitable for enthusiasts and workstation builds. Dimensions: 305mm x 330mm (12" x 13").
